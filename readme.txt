TurnApp_PWA_DeployReady
-----------------------
1) Abrir generate_icons.html -> cargar logo.svg -> descargar icon-192.png y icon-512.png -> guardar en carpeta icons/
2) Subir toda la carpeta al repo de GitHub (o servir desde un servidor).
3) Para PDF del manual: abrir manual.html -> imprimir -> guardar como PDF -> manual.pdf
4) Manifest utiliza icons/icon-192.png y icons/icon-512.png
